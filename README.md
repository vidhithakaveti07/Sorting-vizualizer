# 🌸 SortBloom — Sorting Algorithm Visualizer

✨ *An elegant and interactive way to understand sorting algorithms through visualization.*

SortBloom is a modern web-based sorting visualizer that allows users to explore how different sorting algorithms work step by step with smooth animations, customizable inputs, and intuitive controls.

---

## 📖 Overview

Sorting algorithms are fundamental to computer science, and understanding their internal working is essential for problem-solving and technical interviews.

SortBloom provides a visually engaging platform where users can observe how elements are compared, swapped, and arranged in real time using different algorithms.

---

## 🎯 Key Features

### 🔹 Supported Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort

---

### 🔹 Interactive Controls

* ⏭ **Next Step / Previous Step** navigation
* ▶️ **Play / Pause execution**
* 🔄 **Reset functionality**

---

### 🔹 Custom Input Options

* Enter your own array (comma-separated values)
* Generate random arrays instantly
* Adjust number of bars dynamically

---

### 🔹 Speed Control

* ⚡ Adjustable animation speed using slider
* Helps in both slow learning and fast execution

---

### 🔹 Visual Representation

* 📊 Bar-based visualization of array elements
* 🎨 Color-coded states:

  * Unsorted
  * Comparing
  * Swapping
  * Pivot
  * Merging
  * Sorted

---

### 🔹 Modern UI/UX

* 🌙 Theme toggle (e.g., Midnight mode)
* Clean, aesthetic, and responsive design
* Smooth animations for better user experience

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling & animations
* **JavaScript** – Logic & interactivity

---

## ⚙️ How It Works

1. Choose a sorting algorithm
2. Input a custom array or generate a random one
3. Adjust speed and number of elements
4. Start visualization using play or step controls
5. Observe how the algorithm sorts step by step
